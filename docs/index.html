Simple CRUD Operation

Github Repository

This database library uses PDO(PHP Data Objects) to connect and prepared statement to execute database query. In this Database CRUD library there are 4 classes. Create, Read, Update and Delete.

Below is how to use it

Create

The 1st function in this CRUD is the C named Create. You will use Create to add/insert new data into your database. In order to use it, u must call create($data, $table) function

Create function requires 2 parameters in order for it to work. 

@param = array $data // must be in array format
@param = string $table // must be in string format

You must specify which column and what data you want to be inserted in array $data while in string $table, you must specify the table name that contains the column u specify in array $data

Pass these 2 parameter into create() function in order for it to work properly. 

$data = array('column1' => 'data1');
$table = 'test'; // this is table name

create($data, $table); // this will create new row in database

Or you can also pass the array or string directly into the function like so. 

create(array('column1' => 'data1'), 'test'); // this will create new row in database

both way will work and the generated query will be INSERT INTO 'test'(column1') VALUES ('data1');

Read

The 2nd function is the R named Read. You use Read to get data from database and display it in your html or view. Call read($data, $table) in order to use it.

Read function also requires 2 parameters in order for it to work

@param = array $data // must be in array format
@param = string $table // must be in string format

As you can see, the Read parameters require exact same data type as Create

But in Read you dont need to specify what data u want to read, only column. You can also specify 1 more thing in $data which is a where clause in database query. That way you can choose to read all data or only specific data from your database table

to add a where clause, u must add 'Where' => 'column name' = 'data' into the last part of array $data

Pass these 2 parameter into read() and then call it to use it. 

$data = array(
	'column1', // use * if u want to read all data
	'WHERE' => 'id' = '1'); // this is not required
$table = 'test'; // this is table name

read($data, $table); // this will read column1 data from row id = 1

or do it the 2nd way 

read('column1', 'WHERE' => 'id' = '1', 'test'); // this will read column1 data from row id = 1

The generated query will be SELECT 'column1' FROM 'test';

Update

The 3rd one is the U derived from Update. This is the function you will be using to update your old data in your database to a new one. Call update($data, $id, $table) in order to use it.

Update function is abit different. It requires 3 parameters in order for it to work

@param = array $data // must be in array format
@param = string $id // must be in string or integer format
@param = string $table // must be in string format

Update require 3 parameters because u will have to specify which row u want to be updated or else all row will be affected.

pass all 3 paremeters like so. 

$data = array('column1' => 'data1'); // SET column1 to data1
$id = '1'; // this is id in where clause
$table = 'test'; // this is table name

update($data, $id, $table); // this will update row 1 column1 old data to data1

Or you can also pass the array or string directly into the function like so. 

update(array('column1' => 'data1'), '1', 'test'); // this will update row 1 column1 old data to data1

The query for this function will be UPDATE 'test' SET 'column1' = 'data1' WHERE 'id' = '1';

Delete

This is Delete,the last function of CRUD. You use this to delete any row from any table in your database

Delete function also requires 2 parameters like Create and Read. but one of the parameter is different.

@param = string $id // must be in string or integer format
@param = string $table // must be in string format

Delete function is abit simple because it doesnt require array variables. 

$id = '1';
$table = 'test'; // this is table name

delete($data, $table); // this will delete row with an id of 1 in table test

just like the other function, Delete also able to use a straightforward way as this. 

delete('1', 'test'); // this will delete row with an id of 1 in table test

The generated query will be Delete FROM 'test' WHERE id = '1';
